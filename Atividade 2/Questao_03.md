# Questão 03

Pesquise outros "modelos como serviço" diferentes dos três modelos tradicionais (IaaS, PaaS, SaaS).

a. Descreva o modelo
b. Desenhe uma figura que represente o modelo (ou encontre uma figura que o represente - com o link)
c. Exemplifique seu uso

---

### 1. CaaS (Containers as a Service)

Descrição: Plataforma gerenciada para implantar, orquestrar e escalar contêineres (ex: Docker), abstraindo infraestrutura subjacente; provedor gerencia cluster Kubernetes.
Exemplo de uso: AWS EKS para rodar microsserviços NestJS em contêineres Docker, escalando automaticamente com tráfego de apps de delivery.

### 2. FaaS (Function as a Service)

Descrição: Computação serverless onde você implanta funções executadas sob demanda, sem gerenciar servidores; provedor cuida de escalabilidade e billing por uso.
Exemplo de uso: AWS Lambda para processar uploads em S3, integrando com React frontend em apps de food delivery, cobrando só por execução.
